# Smart Battery Billing System

A desktop application for electronics shops managing battery and inverter sales.

## Features
- QR Code Scanning
- Dynamic Pricing
- Old Battery Exchange
- PDF Invoice Generation
- WhatsApp Integration

## Setup
1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Run the application:
   ```
   python main.py
   ```
